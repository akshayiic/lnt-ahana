# Amenities Static Images

Place your amenity images in this directory with the following filenames:

- `Badminton_Piclkleball court.jpg` - Badminton/Pickleball Court
- `Banquet 300 01.jpg` - Banquet
- `Bussines Center.jpg` - Business Center
- `Gym 300 01.jpg` - Gym
- `Indoor Games 300 01.jpg` - Indoor Games
- `Kids Creche.jpg` - Kids Creche
- `Lobby 300 01.jpg` - Lobby
- `Lounge 300 01.jpg` - Lounge 1
- `Lounge 300 02.jpg` - Lounge 2
- `Yoga 300 01.jpg` - Yoga

The application will automatically display these images when users click on the corresponding menu items in the sidebar.

**Menu Structure:**
- Simple flat list (no categories/accordions)
- Direct mapping to image files
- Centered vertically on screen
- Click any item to display the corresponding image
